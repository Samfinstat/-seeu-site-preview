# Интеграции SeeU — следующие этапы

## Автоматическая ссылка записи

Сейчас мастер скачивает приложение SeeU, регистрируется, копирует индивидуальную ссылку на запись из своего профиля и вручную вставляет её в бриф создания сайта.

После завершения новой версии приложения нужно:

1. связать аккаунт приложения с аккаунтом создания сайта;
2. получать индивидуальную ссылку мастера через защищённый API;
3. автоматически подставлять ссылку в поле `appBookingLink` после регистрации или авторизации;
4. разрешить пользователю проверить и подтвердить ссылку перед публикацией;
5. синхронизировать последующие изменения ссылки без ручного редактирования сайта;
6. определить обработку случая, когда у пользователя несколько профилей или филиалов;
7. вести журнал изменений и не раскрывать токены приложения в браузере.

До подключения API поле остаётся обязательным и заполняется пользователем вручную.
